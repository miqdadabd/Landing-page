Kebijakan Privasi

Terakhir diperbarui: 14 April 2026 · Versi 2.0

Digital Ally ("kami", "milik kami") berkomitmen melindungi privasi Anda. Dokumen ini menjelaskan secara transparan data apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak-hak Anda sebagai pengguna layanan kami, termasuk aplikasi Jurnal Guru dan EduTrack.

1. Data yang Kami Kumpulkan

Kami hanya mengumpulkan data yang diperlukan untuk mengoperasikan layanan:

  • Alamat Email — diperoleh dari akun Google Anda, digunakan sebagai identifikasi unik akun.
  • Nama Lengkap — diperoleh dari profil Google, digunakan untuk personalisasi tampilan dalam aplikasi.
  • Foto Profil Google — ditampilkan di antarmuka aplikasi jika tersedia (tidak disimpan di server kami).
  • Data Aktivitas Mengajar — absensi, kasus siswa, jadwal, catatan anekdot, dan buku jurnal yang Anda masukkan sendiri ke dalam aplikasi. Data ini hanya Anda yang bisa akses.

Kami tidak mengumpulkan: nomor telepon, data pembayaran, lokasi GPS, atau data perangkat keras.

2. Cara Kami Menggunakan Data

  • Mengautentikasi identitas Anda saat masuk ke aplikasi.
  • Menyimpan dan menyinkronkan data pekerjaan Anda ke cloud (fitur Jurnal Guru & EduTrack).
  • Menampilkan nama Anda di halaman sambutan dan laporan yang Anda buat sendiri.
  • Mengirimkan pengumuman penting terkait layanan (hanya via aplikasi, bukan email marketing).

3. Layanan Pihak Ketiga

Kami menggunakan layanan-layanan berikut untuk mengoperasikan platform ini:

  • Google OAuth 2.0 — untuk proses login. Kami hanya menerima data profil dasar (email & nama). Kebijakan privasi Google: policies.google.com/privacy
  • Supabase — penyimpanan database terenkripsi. Data disimpan di server Supabase yang berlokasi di Singapore (ap-southeast-1). Kebijakan privasi Supabase: supabase.com/privacy
  • Vercel — hosting aplikasi web. Kebijakan privasi Vercel: vercel.com/legal/privacy-policy

4. Penyimpanan & Keamanan Data

  • Data aktivitas mengajar Anda disimpan secara terenkripsi di infrastruktur Supabase.
  • Akses data dilindungi oleh Row Level Security (RLS) — hanya akun Anda yang bisa membaca & menulis data Anda.
  • Koneksi ke server menggunakan HTTPS/TLS (terenkripsi end-to-end).
  • Kami tidak pernah menyimpan password — autentikasi sepenuhnya ditangani oleh Google.

5. Retensi Data

Data Anda disimpan selama akun Anda aktif. Jika Anda meminta penghapusan akun, seluruh data yang terkait akan dihapus secara permanen dari server kami dalam waktu 30 hari kerja. Data di localStorage browser Anda dapat Anda hapus sendiri kapan saja melalui pengaturan browser.

6. Hak-Hak Anda

Sebagai pengguna, Anda berhak untuk:

  • Akses — meminta salinan data yang kami simpan tentang Anda.
  • Koreksi — memperbarui data yang tidak akurat (bisa dilakukan langsung di aplikasi).
  • Portabilitas — mengunduh semua data Anda dalam format JSON melalui fitur "Backup Data" di menu Setelan.
  • Penghapusan — meminta agar akun dan seluruh data Anda dihapus permanen.

Untuk menggunakan hak-hak di atas, hubungi kami di: miqdadabdussalam08@gmail.com

7. Cookie & Penyimpanan Lokal

Aplikasi kami menggunakan localStorage browser (bukan cookie) untuk menyimpan preferensi tampilan dan data offline. Tidak ada cookie pelacakan (tracking cookies) atau cookie iklan yang digunakan. LocalStorage dapat Anda hapus setiap saat melalui browser.

8. Anak-Anak

Layanan kami ditujukan untuk guru dewasa. Kami tidak secara sengaja mengumpulkan data pribadi dari individu di bawah usia 13 tahun. Data siswa yang dimasukkan ke aplikasi adalah tanggung jawab guru/sekolah yang bersangkutan.

9. Perubahan Kebijakan

Jika kebijakan ini diperbarui secara material, kami akan memberitahu melalui notifikasi dalam aplikasi setidaknya 7 hari sebelum perubahan berlaku. Penggunaan berkelanjutan setelah perubahan berlaku dianggap sebagai persetujuan terhadap kebijakan baru.
